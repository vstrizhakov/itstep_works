Visual Studio Image Library: 24bit Outline Images

24bit Art
This art is 24bit color. Transparency should be achieved by mapping RGB:255,0,255 to the background color of the UI.

Use Restrictions
Images in this folder are typically used in tree controls (hierarchical structures). The names of most of the elements are self-descriptive, and indicate exactly the element that the image represents. As a part of a visual language, the elements must be used in a manner consistent and identical with each of the names as they appear below:
